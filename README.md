## 1일차

### 1. 자바 프로그래밍 시작하기

- **01-1 프로그래밍과 자바**: 프로그래밍 기본 개념, 자바 언어의 특징
- **01-2 자바 개발 환경 설치하기**: JDK 설치, IDE 설정 (이클립스 설치 등)
- **01-3 이클립스로 첫 프로그램 만들기**: "Hello, World!" 출력 프로그램 만들기

- **02-2 변수란 무엇일까?**: 변수 선언, 초기화
- **02-3 변수가 저장되는 공간의 특성, 자료형**: 기본 자료형 (int, float, char 등)
